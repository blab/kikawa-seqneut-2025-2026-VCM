# kikawa-seqneut-2025-2026-VCM

Nextstrain analysis of Kikawa et al. neutralization assay experiments for 2025-2026 Northern Hemisphere VCM.
This workflow uses publicly available data provided by GenBank.

## Run the workflow

Run the workflow with the default Nextstrain Docker image.

``` bash
nextstrain build --docker .
```

## View the trees

View the trees locally with Auspice.

``` bash
nextstrain view --docker auspice
```

Or view [the H1N1pdm HA tree](https://nextstrain.org/groups/blab/kikawa-seqneut-2025-2026-VCM/h1n1pdm?d=tree,measurements,frequencies&f_kikawa=present_1&p=grid) or [the H3N2 HA tree](https://nextstrain.org/groups/blab/kikawa-seqneut-2025-2026-VCM/h3n2?d=tree,measurements,frequencies&f_kikawa=present_1&p=grid) in the "blab" Nextstrain Group.
